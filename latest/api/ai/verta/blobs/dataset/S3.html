case class S3(contents: HashMap[String, FileMetadata], enableMDBVersioning: Boolean = false) extends Dataset with Product with Serializable

Captures metadata about S3 objects. Please set up AWS_ACCESS_KEY_ID, AWS_SECRET_ACCESS_KEY, and AWS_REGION environment variables before use. To create a new instance, use the constructor taking a list of S3 Locations or a single location

val s3Blob: Try[S3] = S3(List(S3Location("some-path-1").get, S3Location("some-path-2").get))
val s3Blob2: Try[S3] = S3(S3Location("some-path"))
